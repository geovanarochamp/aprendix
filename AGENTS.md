# Regras do Aprendix

Ao criar ou revisar exercícios, usar a skill pessoal `$aprendix-revisar-atividades` quando disponível.

Regra didática obrigatória: quando os elementos exibidos no enunciado forem as próprias respostas candidatas, as alternativas devem corresponder exatamente a eles. Dois candidatos exibidos exigem duas opções; três candidatos exigem três opções; não adicionar opções que não apareceram. Distratores extras continuam válidos em contas, contagens, lacunas, sequências, códigos e outras tarefas nas quais o conteúdo exibido não representa as alternativas candidatas.

Revisar cada questão individualmente para uma criança de aproximadamente 7 anos. Garantir enunciado direto, uma única resposta inequívoca, correspondência entre imagem e texto, divisão silábica correta, resposta presente nas opções e ausência de opções duplicadas. Em lacunas de letras ou sílabas, usar exatamente um traço por letra esperada (`__` para GA, `___` para SOL). Após mudanças, executar o build e conferir visualmente uma questão representativa no mobile.
