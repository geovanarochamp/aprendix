# Aprendix

Protótipo web de um app infantil multidisciplinar para crianças do 1º ano.

## Stack

- React + TypeScript
- Vite
- Tailwind CSS
- Framer Motion
- Zustand
- Canvas para caligrafia
- PWA instalável

## Como rodar

```bash
npm install
npm run dev
```

Depois acesse `http://localhost:5173`.

## Scripts

```bash
npm run dev
npm run build
npm run preview
```
